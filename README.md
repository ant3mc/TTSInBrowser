  TTS (синтез речи)  для голосов, доступных в браузерах ( к примеру Edge, в том числе Multilingual, CHrome и т.д.). Функционал:  загрузка txt и html, вставка текста, настройки, словарь и т.д. Можно удобно управлять с клавиатуры. 
   Реализована комплексная защита от XSS-атак при загрузке файлов и вводе текста.
 
  All-in-one index.html file( CSS and JS полностью интегрированы в html-файл). 
 Нужно просто скачать этот html и открыть его в любом браузере.
 
![IMG_20260227_202510](https://github.com/user-attachments/assets/c43afbc5-0bd6-4d94-9dcc-6a0b27dc8f78)

 В мобильном Edge **сразу** после загрузки( или перезагрузки) приложения нужно активировать голоса. Для это нужно выделить слово в названии приложения и вызвать встроенную функцию браузера "прочесть вслух". После этого можно выбирать голоса в самом приложении.
  Чтобы озвучка продолжалась на мобильных устройствах с выключенным экраном, нужно отключить энергосбережене у браузера.
 
  Чтобы с клавиатуры перейти из режима "ввод текста" в простой режим озвучки( где не нужны комбинации с Alt) надо нажать Tab (выведя фокус на элемент управления вверху приложения).
